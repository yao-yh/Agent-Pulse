# 代理配置

AgentPulse 的本地代理路径是：

```text
http://127.0.0.1:8080/proxy/{proxyKey}
```

集成适配器会为不同工具生成代理配置计划。计划应用成功后，AgentPulse 会记录原始上游地址到本地代理地址的映射。

## 安全流程

配置变更必须支持：

1. plan：生成计划，展示目标文件和变更意图。
2. backup：写入前备份原文件和元数据。
3. apply：应用配置变更。
4. rollback：按备份恢复或删除本次创建的配置文件。

## 流式响应

代理层保持 SSE、chunked、未知长度响应的直接透传，不为了摘要记录阻塞用户侧响应路径。
