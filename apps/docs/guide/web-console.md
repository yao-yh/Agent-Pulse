# Web 控制台

Web 控制台默认位于 `http://127.0.0.1:8080`。

## Agents

扫描 Codex、Claude Code、OpenCode 等工具的配置来源，显示是否已经路由到 AgentPulse 本地代理。

替换配置前，页面会展示目标文件、原始上游地址、新代理地址和备份说明。确认后才会写入用户级配置。

## Events 和 Tasks

展示 hook 事件、任务和会话记录。事件会被基础分析器标记风险级别，敏感字段默认脱敏。

## Inventory

扫描本机 skills、MCP servers、plugins 和配置来源。扫描结果写入本地存储，不保存明文密钥。

## Proxy

展示通过本地代理捕获的请求摘要。流式响应会直接透传，避免阻塞用户侧响应。

## Doctor

汇总健康状态、集成扫描结果和近期任务，便于排查本地环境问题。
